How to start syncing

  1. Save the config above.
  2. Open a terminal in the project folder and run: 
    python sync_watcher.py
  3. Leave it running. Every time you add a song to your Spotify playlist, it will be found on YouTube Music and added automatically.

To run it silently in the background on Windows you can use Task Scheduler or pythonw sync_watcher.py.
